# Sanjeevni Infra - Production Static Website

Premium multi-page corporate website built with plain HTML, CSS, and JavaScript.

## Pages

- `index.html` - Home and company overview
- `projects.html` - Project portfolio with category filters
- `blog.html` - Blog/news listing
- `careers.html` - Open jobs and apply-by-email actions
- `contact.html` - Inquiry form with email integration + contact details

## Shared Assets

- `css/style.css` - Global design system and responsive layout
- `js/data.js` - Default content and local storage persistence
- `js/main.js` - Navigation/mobile menu/shared UI behavior
- `js/projects.js` - Project rendering and filtering
- `js/blog.js` - Blog/news rendering
- `js/careers.js` - Careers rendering
- `js/contact.js` - Inquiry form email workflow

## SEO

- `robots.txt`
- `sitemap.xml`
- Page-level meta descriptions, canonical tags, and basic Open Graph setup

## Run

Open any page directly, or use a static server for best behavior.
